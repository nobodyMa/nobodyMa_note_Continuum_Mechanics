## 课程简介

《连续介质力学》是力学、物理、材料科学和工程学科的核心基础课程，系统研究宏观尺度下物质运动和变形的普适性规律。赵亚溥老师的课程以深厚的理论功底和学科前沿视角，贯通经典力学框架与现代交叉应用，培养从连续介质角度建模和分析复杂物理现象的能力。
